# Web content specs

Concrete limits and targets for quick reference. Only the numbers below are sourced; don't add
others without a source. These are guardrails, not goals — match depth and length to what the
reader's question actually needs.

## Metadata

Title-tag, meta-description, and Open Graph character limits are technical SEO concerns outside
this rubric — don't flag them. The writing-side rules:

| Thing | Target |
| --- | --- |
| Slug | short, lowercase, hyphenated, keyword in it, no dates/stop words |
| Title content | lead with the key information; keyword near the front |

## Headings & structure

| Thing | Target |
| --- | --- |
| H1 per post | exactly one (the title) |
| Heading hierarchy | H1 → H2 (sections) → H3 (sub-points); don't skip levels |
| Max text before a new subheading | ~300 words |
| Paragraph length | one idea, 2–4 sentences, under ~150 words |

## Readability

| Thing | Target |
| --- | --- |
| Sentence length | most under ~20 words; keep ≥20-word sentences under ~25% |
| Passive voice | under ~10% of sentences |
| Words read on an average page | ~28% (people scan, ~79% scan vs ~16% read fully) |

## Word count by intent

No magic number — Google says word count isn't a ranking factor. Benchmark against the current
top results for the query and cover the topic at least as fully, then stop.

| Intent / type | Rough range |
| --- | --- |
| Quick answer / simple query | 300–500 words |
| Standard informational post / how-to | ~1,500–2,500 words |
| Competitive or pillar guide | 2,000+ words, as the subject earns it |
| Yoast thin-content floor (regular page) | ~300 words minimum |
| Yoast cornerstone-content floor | ~900 words minimum |

## Images

| Thing | Target |
| --- | --- |
| Alt text length | ~125 characters or fewer (screen readers cut off around there) |
| Alt text content | describe the image in context; keyword in ≥1, no stuffing |
| Phrasing | skip "image of"/"picture of"; end with a period |
| Decorative images | empty `alt=""` so screen readers skip them |

## Internal links

| Thing | Target |
| --- | --- |
| Contextual internal links per post | ~3–5, placed early in the body |
| Anchor text | descriptive and varied; never "click here" |
